# Projeto de Algoritmos Genéticos para o Problema da Mochila

## Descrição
Este é um projeto que utiliza Algoritmos Genéticos para resolver o clássico Problema da Mochila. O Problema da Mochila é um problema de otimização combinatória que envolve selecionar um conjunto de itens de uma lista, cada um com seu valor e peso, de forma a maximizar o valor total dos itens selecionados, sem exceder a capacidade máxima de peso de uma mochila.

## Pré-requisitos
- Python 3.x
- Bibliotecas de computação evolutiva, como DEAP (Distributed Evolutionary Algorithms in Python)
